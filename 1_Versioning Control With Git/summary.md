(1) Versioning Control With Git
    - Versioning digunakan untuk mengatur versi dari source code program, dimaan terdaat 3 Version Control System yaitu Single User, Centralized, dan Distribute.
    - Git adalah salh satu Version Control System dengan github sebagai layanan hosting git.
    - Terdapat 3 branch yang dapat digunakan agar kolaborasi dalam pengembangan software dapat berjalan dengan optimal yaitu Master, Feature, dan Develop.
